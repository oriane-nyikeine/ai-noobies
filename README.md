# AI Noobies

WordPress site making AI accessible for beginners, covering tools, tutorials and practical use cases for people discovering artificial intelligence.

Co-founder of the project. I handled the full technical and design side: installation, configuration, theme, structure, optimisations. My partner was in charge of the editorial direction and content management.

## What I did

- Full WordPress site setup (installation, configuration, structure)
- Theme selection and configuration (Newscard)
- Plugin configuration (SEO, performance, security)
- Ongoing maintenance and updates

## Stack

**CMS** — WordPress
**Hosting** — O2Switch
**Domain** — ainoobies.net
